# Unit 6 — CSS Selectors Practice (3/18)
CSS practice: Writing class and ID rules and applying them in HTML.

---

> 💡 Start in `styles.css` first — then open `index.html`

---

## Part 1 — Fully Guided

We give you the rule name AND what goes inside.
Your job is to write the syntax correctly.

---

**In `styles.css`:**

**Task 1** — Write a class rule called `pink-bg`
- Change the background color to pink

**Task 2** — Write a class rule called `blue-text`
- Change the text color to blue

**Task 3** — Write a class rule called `big-border`
- Add a border using: `3px solid black`

**Task 4** — Write an ID rule called `main-title`




**Task 5** — Write an ID rule called `footer-text`
- Change the text color to gray
- Change the background color to black

---

**Now open `index.html` and apply your rules:**

- Add `class="pink-bg"` to the first paragraph and the last two random text paragraphs
- Add `class="blue-text"` to the second paragraph
- Add `class="big-border"` to the third paragraph and all five images
- Add `id="main-title"` to the `<h1>`
- Add `id="footer-text"` to the `<p>` inside the footer

---

## Part 2 — Semi Guided

We give you the rule name but NOT what goes inside.
You decide the properties and values.
Each rule must have at least `background-color` and `color`.

---

**In `styles.css`:**

**Task 6** — Write a class rule called `highlight`

**Task 7** — Write a class rule called `list-style`

**Task 8** — Write an ID rule called `page-heading`

---

**Now open `index.html` and apply your rules:**

- Add `class="highlight"` to the `<ul>` list
- Add `class="list-style"` to the `<ol>` list
- Add `id="page-heading"` to the second `<h2>`

---

## Part 3 — Open Ended

You decide everything — the name AND the styles.
Write **2 class rules** and **1 ID rule** in `styles.css`.
Then apply them anywhere you want in `index.html`.

Good luck! 🎯
